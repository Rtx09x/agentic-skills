# v0.1.0

Initial release of Agentic Skills.

## Included Skills

- `$spec`: converts rough intent into implementation-ready specs with task trees, acceptance criteria, verification plans, and `$ship` handoffs.
- `$ship`: executes end-to-end with high agency: inspect, implement, work through blockers, verify, polish, and report.

## Install

Windows:

```powershell
git clone https://github.com/Rtx09x/agentic-skills.git
cd agentic-skills
.\scripts\install.ps1
```

macOS / Linux:

```bash
git clone https://github.com/Rtx09x/agentic-skills.git
cd agentic-skills
./scripts/install.sh
```

## Notes

This release focuses on the two core operating modes:

- spec deeply,
- ship relentlessly.
